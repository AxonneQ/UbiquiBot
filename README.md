UbiquiBot is an open source multi-purpose discord bot.

Main features include:

* Random Jokes
* Server Statistics
* Experience System for Members

Planned Features:

* Music features
* Song Lyrics search
* Hangman game
* Blackjack
* Points system
* Reaction Based Role Assignment
* Voting System

... and more!
