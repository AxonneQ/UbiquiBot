UbiquiBot is an open source multi-purpose discord bot.

Main features include:

* Random Jokes
* Server Statistics
* Experience System for Members (including messages since beginning of the server.)

Planned Features:

* Music features
* Hangman game
* Text Based RPG Game
* Reaction Based Role Assignment

... and more!
